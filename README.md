# API de Bandeira Tarifária 

API de bandeira tarifária desenvolvida em Spring Boot que realiza web scrapping e exibe os dados em JSON.

### Requisitos

* JDK 17
* Docker

### Tecnologias

* Java 17
* Spring Boot 3
* PostgreSQL 

### Variáveis de ambiente
```
DB_URL
DB_USERNAME
DB_PASSWORD
DB_SCHEMAS
SOURCE_URL
SCHEDULER_INTERVAL
```
